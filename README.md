
Este projeto é um sistema simples de cadastro e gerenciamento de veículos, criado para fins acadêmicos e também para demonstrar habilidades práticas em:

 Lógica de programação
 Consumo de APIs
 Estruturação de fluxos internos
 Desenvolvimento front-end
 Integração com backend via JSON Server



 Funcionalidades:

CRUD
 Cadastrar veículos  
 Listar todos os veículos cadastrados  
 Remover veículos  
 Status “Disponível” ou “Vendido”  
 Interface simples e objetiva  
 Dados persistidos em API (JSON-Server)

 Tecnologias Utilizadas

HTML5
CSS3
JavaScript
